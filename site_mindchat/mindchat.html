Projeto

Como funciona o MindChat ? O Venom Bot é uma biblioteca que permite a automação do WhatsApp Web. Ele é usado para criar uma instância do WhatsApp em um navegador e interagir com as mensagens recebidas e enviadas.

O MindChat inicia o Venom Bot, que exibe um código QR no terminal. O usuário escaneia esse código usando o WhatsApp no seu dispositivo móvel, fazendo login no WhatsApp Web.

Quando o usuário envia uma mensagem para o número do WhatsApp associado ao Venom Bot, o MindChat processa essa mensagem utilizando o Dialogflow para gerar uma resposta apropriada.

A resposta é então enviada de volta ao usuário via WhatsApp usando o Venom Bot.
